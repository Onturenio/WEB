Git repositories

I have uploaded several repositories with various pieces of (quite simple) software to Github.

  • MM5toNETCDF is a software package to post-process the outputs of the MM5 model and create a number of diagnosed variables in netCDF format. It requires netCDF libraries.
  • BeamerTemplate is a template for the LaTeX clase Beamer to create presentations. This template basically modifies its final aspect to make it look less LaTeX-like. It is not a theme itself, but rather a set of commands in the preamble to redefine how Beamer behaves.
  • WEB is a repository with a copy of the source code of this personal web.

The get the code, install git and then run the command:

git clone <repository_url>

You will get an exact copy of the repository, with all the files and the .git directory with the information to work with git (if you are interested).

My .vimrc file

You never know when you are going to have to work in a computer where you can not get access to a copy of your .vimrc, so here is a copy of it... just in case.

set nocompatible
syntax on
set number
set ruler
set autoindent
set smarttab
set tabstop=2
set softtabstop=2
set shiftwidth=2
set showmatch
set vb
set incsearch
set backspace=indent,eol,start
set expandtab
:let fortran_free_source=1
set hls
set mouse=a
map <UP> gk
map <DOWN> gj
imap <UP> <ESC>gka
imap <DOWN> <ESC>gja
set ww=h,l,<,>,[,],s,
vnoremap <BS> :s/^ *//<CR>
map <c-d> <leader>c<space> 
"inoremap ( ()<left>
"inoremap { {}<left>
"inoremap [ []<left>
" Enables F2 as a hotkey to enter in Paste mode (this is usefull to copy text
" from other programs through the mouse)
nnoremap <F2> :set invpaste paste?<CR>
set pastetoggle=<F2>
set showmode
set viewoptions+=cursor,folds
